DATABASE QUERIES

View pending approvals in database:
  python3 << 'EOF'
  import sqlite3
  db = sqlite3.connect('backpocket.db')
  cursor = db.cursor()
  cursor.execute("SELECT ref_id, sender, status FROM pending_approvals LIMIT 5")
  for row in cursor.fetchall():
      print(f"{row[0]}: {row[1]} - {row[2]}")
  db.close()
  EOF

  import sqlite3
  db = sqlite3.connect('backpocket.db')
  cursor = db.cursor()

  db.close()
  EOF

COMMON TESTING WORKFLOWS
================================================================================

Workflow 1: Test Complete Approval Cycle (2 min)
  1. curl -s http://127.0.0.1:8000/api/pending | jq 'length'
     → Check initial count (5)
  2. curl -X POST http://127.0.0.1:8000/api/approve -H "Content-Type: application/json" \
       -d '{"ref_id": "DEMO-00001"}'
     → Approve one item
  3. curl -s http://127.0.0.1:8000/api/pending | jq 'length'
     → Verify count decreased (4)

Workflow 2: Test Draft Editing (3 min)
  1. curl -s http://127.0.0.1:8000/api/draft/DEMO-00002 | jq .draft_body
     → See original draft
  2. curl -X POST http://127.0.0.1:8000/api/save-draft \
       -H "Content-Type: application/json" \
       -d '{"ref_id": "DEMO-00002", "draft_body": "Updated response text"}'
     → Save new draft
  3. curl -s http://127.0.0.1:8000/api/draft/DEMO-00002 | jq .draft_body
     → Verify updated

Workflow 3: Test Document Processing (5 min)
  1. echo "Invoice from Supplier" > test_invoice.txt
     → Create test document
  2. curl -X POST http://127.0.0.1:8000/api/documents/upload \
       -F "file=@test_invoice.txt" -F "document_type=invoice"
     → Upload document
  3. curl -s http://127.0.0.1:8000/api/documents | jq '.[-1]'
     → See uploaded document
  4. curl -X POST http://127.0.0.1:8000/api/documents/analyze/<doc_id>
     → Start analysis (Moondream)
  5. (wait 30 sec) curl -s http://127.0.0.1:8000/api/documents/<doc_id> | jq .analysis
     → Check results
